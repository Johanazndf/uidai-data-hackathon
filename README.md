## Overview
This project was developed as part of the UIDAI Data Hackathon.  
The objective is to analyze and visualize Aadhaar-related datasets to identify trends, gaps, and patterns in enrolment, demographic coverage, and biometric capture across regions.

The project focuses on **data-driven insights**, not raw data exposure.

---

## Problem Statement

Unlocking Societal Trends in Aadhaar Enrolment and Updates

Identify meaningful patterns, trends, anomalies, or predictive indicators and translate them into clear insights or solution frameworks that can support informed decision-making and system improvements.

---

## Approach
- Preprocessed Aadhaar datasets to remove noise and inconsistencies
- Performed region-wise and time-based aggregation
- Generated visual insights to highlight:
  - Enrolment gaps
  - District and state-level trends
  - Temporal variations in Aadhaar capture
